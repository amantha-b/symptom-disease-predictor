Features

- Predicts a probable disease based on selected symptoms

- Displays the top three most likely diseases with probability scores

- Pre-trained model loads instantly using .pkl files

- Clean, modern, fully customized UI with CSS and theme overrides

- Highly modular and easy to extend or retrain

Web Framework
